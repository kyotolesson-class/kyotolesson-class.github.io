Commande Linux

  • Start a Terminal on Debian distribution use simultaneous CTRL + ALT + T

  • ls List the contents of a directory (or folder).

  • pwd Displays the path of the current folder.
  • cd Allows you to walk around the folders.
  • mkdir Create a new directory.
  • touch Create a new file.
  • cp Copy files or directories.
    • cp file1 file2 Here, copy the contents of file1 into file2.
    • cp folder1 / file1 folder2 Here, we copy file1 in folder2.
    • cp folder1 / file1 folder2 / file2 folder3 / Here, we copy file1 and file2 in folder3.
  • mv Move or rename a file or folder.
    • mv file1 file2 folder1 / Here, we move file1 and file2 in folder1.
    • mv file1 file2 Here, we rename file1 to file2.
  • rm Delete a file.
    • rmdir remove a folder.
  • cat Displays the contents of a file.
    • cat file1> file2 Here, all the contents of file2 are replaced by the contents of file1.
    • cat file1 >> file2 Here, the contents of file1 are added to the contents of file2. | This is the "pipe", it passes the output of a command inpout another command.
    • cat file1 | Here, the output of cat becomes the inpout of wc.
