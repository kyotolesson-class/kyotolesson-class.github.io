Commande Linux

Start a Terminal on Debian distribution use simultaneous CTRL + ALT + T

If I say:

Create a Folder or a Directory called or named test

You should be able to do...

root@hack:~/Desktop# mkdir test

Create a File called or named ABC

You should be able to do...

root@hack:~/Desktop# touch ABC

List or Check me the content of your folder

You should be able to do...

root@hack:~/Desktop# ls

Go to or Go inside folder XYZ

You should be able to do...

root@hack:~/Desktop# cd XYZ

Change the name or rename the FILE or the FOLDER Pikachu to Raichu

You should be able to do...

root@hack:~/Desktop# mv Pikachu Raichu

mv is use the name for File and Folder

Delete or Remove a File named ABC

You should be able to do...

root@hack:~/Desktop# rm ABC

Delete or Remove a Folder named XYZ

You should be able to do...

root@hack:~/Desktop# rmdir XYZ
