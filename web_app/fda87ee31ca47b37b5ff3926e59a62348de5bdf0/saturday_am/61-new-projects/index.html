Need help getting started? Try this guide:

  1. Drag the image you want to edit to the browser window.


  2. Use the "color picker" tool to select your base color.


  3. Choose the brush tool. This is what we use to draw!


  4. Change your brush size to something more useful. For this, I want a nice, big 50-point brush.


  5. Draw!!!
