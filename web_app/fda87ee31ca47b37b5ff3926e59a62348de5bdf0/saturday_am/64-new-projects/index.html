Everyone's making steady progress on their individual work. Something interesting came up for multiple students that deserved its own lesson:

How do you change an image on a page simply by clicking on it?

Here's how we do it:

  1. Insert the image onto the page using a standard img tag, like so:
    <img src="...">

  2. In JavaScript, select the image like this:
    document.querySelector('img')

  3. Since we're clicking on the image to change it, we'll want an onclick function:
    document.querySelector('img').onclick = function(){}

  4. Inside the function, we'll select our image again, but this time we'll change the src attribute to our new image:
    document.querySelector('img').src = "..."
