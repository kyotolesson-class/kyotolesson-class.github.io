I think everyone in the class is now ready to start adding features, images, videos, and simple styles to their own person websites.

GitHub Recap

In case you couldn't set up your repo on your computer, try following these steps:

  1. Check your email for an invitation to your GitHub repo from "kyotolesson-class."
  2. Click the green button that says "View invitation."



  3. Sign into your GitHub account, if necessary.
  4. Inside the repo, click "Code" then click the clipboard to copy the code that starts with git@github.com:.



  5. In your Terminal, cd to your Octocat directory.
  6. Type in git clone, hit space, paste the code you copied from GitHub, then hit enter.



If everything goes well, you should download your repo right into a new directoy in your Octocat folder. Have fun!
